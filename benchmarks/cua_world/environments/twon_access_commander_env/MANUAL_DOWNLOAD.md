# Manual data download

This environment expects the 2N Access Commander VirtualBox OVA, distributed only through 2N's account portal.

## File

| Path | Size |
|---|---|
| `data/access_commander.ova` | ~1.2 GB |

## How to obtain it

1. Sign up at https://my2n.com/ (free account; partner-tier may be needed).
2. Navigate to **Support → Download Center → Software & Firmware → Access Commander**.
3. Download the **VirtualBox / VMware OVA**.
4. Place at `data/access_commander.ova`.

## Verify

```sh
sha256sum data/access_commander.ova
```

Expected sha256:
```
8fa4abfc9d955118321ac7934410f9ee4584374c9539490041d3f2cf0aa6499b
```
