# Animal Shelter Manager (ASM3) Environment Notes

## Application Details
- **App**: Animal Shelter Manager 3 (ASM3)
- **GitHub**: https://github.com/sheltermanager/asm3
- **Type**: Python web application (web.py framework)
- **Database**: PostgreSQL
- **Default Port**: 8080 (web.py default, NOT configurable via asm3.conf)
- **Login**: user / letmein

## Installation Quirks

### 1. Entry Point
The main entry point is `main.py` (in `/opt/asm3/src/`), NOT `code.py` as older documentation suggests.

### 2. __version__.py Must Be Generated
ASM3 requires `src/asm3/__version__.py` with `VERSION` and `BUILD` variables. This is normally generated by `make dist` but must be created manually when cloning from git:
```bash
VERSION=$(cat /opt/asm3/VERSION)
cat > /opt/asm3/src/asm3/__version__.py << EOF
VERSION = "${VERSION} [$(date)]"
BUILD = "$(date +%m%d%H%M%S)"
EOF
```

### 3. lxml Dependency
`lxml` is required by `asm3/checkmicrochip.py` but not included in basic pip requirements. Must install explicitly.

### 4. Config File Format
The config file (`asm3.conf`) uses simple `key = value` format, NOT INI sections. Place at `/opt/asm3/src/asm3.conf` or `/etc/asm3.conf`.

### 5. Web Server Port
ASM3 uses web.py which defaults to port 8080. The `base_url` config setting controls redirect URLs but does NOT change the listening port. To change the port, you'd need to pass `8080` as a command line arg to `main.py`.

## Database Setup

### Schema Initialization
The database schema is initialized via HTTP POST to `/database`:
```bash
curl -X POST http://localhost:8080/database \
  -d "action=create" -d "dbtype=POSTGRESQL" \
  -d "host=localhost" -d "port=5432" \
  -d "username=asm" -d "password=asm" \
  -d "database=asm" -d "locale=en"
```
This takes approximately 80 seconds.

### NOT NULL Constraints
The `animal` table has 58 NOT NULL columns. When inserting records programmatically, ALL of them must be provided with valid values (mostly 0 for integer fields).

## Data Import
Real data from Austin Animal Center (City of Austin Open Data Portal):
- Intakes CSV: Animal ID, Name, DateTime, Found Location, Intake Type, Animal Type, Sex, Age, Breed, Color
- Map fields to ASM3 schema: AnimalTypeID (2=Dog, 11=Cat), SpeciesID (1=Dog, 2=Cat), Sex (0=Female, 1=Male, 2=Unknown)

## Service Timing

3. Database initialization takes ~80 seconds (one-time)
